Ramadan Scientific Library 📚
An advanced web application designed for exploring and managing scientific literature.

Key Features:
Book Discovery: Powered by the Open Library API to search and explore thousands of scientific titles.

Smart Translation: Integrated Arabic translation for book summaries to overcome language barriers.

Watchlist Management: Capability to save favorite books locally and manage them efficiently.

Responsive Design: A clean, scientific-themed UI built with React and custom CSS, fully optimized for Arabic (RTL).

Tech Stack:
Frontend: React (Vite)

Routing: React Router DOM

API: Open Library API & MyMemory Translation API

Styling: Custom CSS with Cairo Font
